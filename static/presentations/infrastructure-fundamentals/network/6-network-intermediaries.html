Infrastructure Fundamentals

What are Network Intermediaries?

Devices or services positioned between clients and servers to manage traffic flow

Perform routing, load distribution, security filtering, caching, and infrastructure abstraction

Enable capabilities that individual clients/servers cannot coordinate alone

Fundamental components of production network architecture

Proxy Servers

Mediates between clients and external servers (outbound traffic)

Client requests go to proxy first; proxy forwards to target and relays response

Functions: Access policy enforcement, content filtering, caching, logging, anonymity

Clients explicitly configure to use the proxy; external servers see proxy IP, not client IP

Reverse Proxies

Sits in front of backend servers, intercepting inbound client requests

Operates transparentlyâ€”clients believe they're connecting directly to the server

Functions: Request distribution, SSL/TLS termination, caching, security filtering

Provides stable interface while backend infrastructure changes (add/remove servers)

Load Balancers

Distribute incoming traffic across multiple servers for performance and availability

Layer 4: Routes based on IP/port; fast, minimal inspection

Layer 7: Routes based on HTTP content (URL, headers, cookies); enables sophisticated rules

If one server fails, load balancer routes to remaining healthy servers

Load Balancing Algorithms

Round-robin: Cycles through servers sequentially; simple, works when servers are equal

Least connections: Routes to server with fewest active connections; handles variable processing times

IP hash: Same client IP always reaches same server; supports session persistence

Weighted: Assigns different weights based on server capacity

Bastion Hosts

Provides controlled access to servers within a private network

Sits in DMZâ€”between public internet and internal network

Administrators connect to bastion first, then to internal servers from there

Hardened: Minimal software, strict firewall rules, logging, multi-factor authentication

Azure Bastion: Managed service providing browser-based SSH/RDP without exposing VMs publicly
