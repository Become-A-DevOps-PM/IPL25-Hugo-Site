Uppdrag (Business Demand)

Webinariet var en succé! Marknadsavdelningen vill nu kunna skicka ut nyhetsbrev för att marknadsföra kommande webinarier.

Lösningen ska innehålla en sida där kunder kan registrera sig för att prenumerera på nyhetsbrevet.

Vi behöver också ett adminområde för att hantera prenumeranter.

Uppdrag (Business Needs)

1. Lösning

Nyhetsbrevslösningen ska hostas på Azure och skrivas i Python/Flask.

2. Registreringsformulär

Ett funktionellt registreringsformulär för att samla in prenumerantuppgifter som namn, e-postadress, företag och jobbtitel. Formuläret ska ha datavalidering.

3. Adminområde

Lösningen ska ge en administratör möjlighet att hantera prenumeranterna.

4. Snabb driftsättning

Lösningen ska ha en CI/CD-pipeline med GitHub Actions för att möjliggöra frekventa driftsättningar.
