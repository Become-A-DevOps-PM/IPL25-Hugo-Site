
# Agile Software Development and Deployment, 25 YH credits

## Course Description

In this course, the student gains knowledge of agile software development and deployment processes. The students shall develop a simple web application with a database that is deployed by creating a CI/CD pipeline. The course also includes an orientation in how virtualization techniques are used in software development.

## Assessment Criteria

Assessment of the student's knowledge and skills is made in relation to learning objectives. It is the student's responsibility to submit assignments on time.

- **Pass (Godkänt):** G required on all components
- **Distinction (Väl godkänt):** G + VG required on all components

## Learning Objectives

| Category | Objective | Pass (G) | Distinction (VG) | Examination |
|----------|-----------|----------|------------------|-------------|
| **Knowledge** | The student can describe fundamental concepts, practices and methods in software development and deployment, as well as an application's architectural design patterns | The student has achieved the objective | — | Individual written assignment |
| **Skill** | The student can drive software development according to agile principles | The student has achieved the objective | The student can describe the process with greater clarity and motivate why, and place concepts in the correct context | Individual written assignment |
| **Skill** | The student has fundamental skills in setting up a simple CI/CD pipeline | The student has achieved the objective | — | Individual written assignment |
