# Server, Network and Storage Technology and IT Security, 30 YH credits

## Course Description

In this course, the student gains knowledge and skills in server, network, and storage technology as well as IT security. Furthermore, the course provides skills in how these technologies are applied in cloud platforms. In the course, the student will also apply their knowledge of IT security by performing risk analyses. The course is conducted as a small project in agile form to focus on the project manager's role in implementing IT infrastructure.

## Assessment Criteria

For a passing grade (G) on the course, G is required on all objectives examined through written assignments.

For distinction (VG) on the course, G is required on all objectives plus VG on objectives that have this criterion. Assessment of the student's knowledge and skills is made in relation to learning objectives. It is the student's responsibility to submit assignments on time.

- **Pass (Godkänt):** G required on all components
- **Distinction (Väl godkänt):** G + VG required on all components

## Learning Objectives

| Category | Objective | Pass (G) | Distinction (VG) | Examination | Assignment |
|----------|-----------|----------|------------------|-------------|------------|
| **Knowledge** | The student can explain how IT systems and their components are structured | The student has achieved the objective | — | Written assignment | **1** |
| **Skill** | The student can build a functioning IT configuration and present its security aspects | The student has achieved the objective | The student can apply their knowledge for a more secure and robust solution of the IT configuration | Written assignment | **1** |
| **Knowledge** | The student can explain concepts within IT security | The student has achieved the objective | — | Written assignment | 2 |
| **Skill** | The student can conduct a risk analysis | The student has achieved the objective | — | Written assignment | 2 |

### Assignment Coverage

- **Assignment 1:** IT infrastructure (servers, networks, storage, cloud deployment)
  - Knowledge: IT systems and components
  - Skill: Build functioning IT configuration with security aspects

- **Assignment 2:** IT security and risk analysis
  - Knowledge: IT security concepts
  - Skill: Conduct risk analysis
