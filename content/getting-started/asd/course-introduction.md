# Course Introduction

[Öppna presentationen](/presentations/course-introduction-asd.html)

## Kursens innehåll

Kursen fokuserar på att bygga och driftsätta en webbapplikation med moderna verktyg och metoder:

- **Vecka 1 - Setup:** Jira, GitHub, lokal Flask-utveckling
- **Vecka 2 - Arkitektur:** Trelagrers-applikation med presentations-, affärslogik- och datalager
- **Vecka 3 - CI/CD:** GitHub Actions för automatiserad driftsättning
- **Vecka 4 - Auth/z:** Autentisering och auktorisering
- **Vecka 5 - Inlämning:** Konsolidering och projektredovisning

### Teknisk stack

- Python 3.11+ med Flask
- Docker för containerisering
- Azure Container Apps för driftsättning
- Azure SQL Database för datalagring
- GitHub Actions för CI/CD
