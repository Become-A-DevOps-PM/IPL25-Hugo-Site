# Project Assignment

[Öppna presentationen](/presentations/project-assignment-asd.html)

## Uppdraget

Efter webinariets succé vill marknadsavdelningen på CM Corp nu ha en lösning för att hantera nyhetsbrev och marknadsföra kommande webinarier.

### Krav

1. **Lösning** - Nyhetsbrevslösning hostad på Azure, skriven i Python/Flask
2. **Registreringsformulär** - Samla in prenumerantuppgifter med datavalidering
3. **Adminområde** - Hantera prenumeranter
4. **CI/CD** - GitHub Actions pipeline för frekventa driftsättningar
