# Week 5: IT Security

This week introduces fundamental IT security concepts and frameworks. You will learn about security principles that apply to all infrastructure and application deployments.

## Theory

Understand the key security frameworks and principles:

- [IT Security](/it-security/) - CIA Triad, Defense in Depth, Zero Trust Architecture, Security by Design, AAA, and IAM

## Practice

Apply security thinking to your work:

- Read through all IT security framework articles
- Analyze your Flask application from previous weeks through a security lens
- Identify potential security improvements based on the frameworks
