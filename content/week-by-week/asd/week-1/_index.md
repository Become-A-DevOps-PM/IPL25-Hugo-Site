# Week 1: Setup

This week focuses on setting up your complete development environment and learning the inner loop workflow. By the end of the week, you will have a working local development setup with Git, GitHub, Jira, and a running Flask application on your laptop.

- [Connect Your Repository to GitHub](/exercises/code-collaboration/2-connect-repository-to-github/) - Push your local repo to GitHub

- [Connect Jira to GitHub](/exercises/code-collaboration/5-connect-jira-github-integration/) - Link commits and PRs to Jira issues
- [Development Workflow with Jira](/exercises/code-collaboration/6-development-workflow-with-jira/) - Complete cycle: story → branch → commit → PR → merge
