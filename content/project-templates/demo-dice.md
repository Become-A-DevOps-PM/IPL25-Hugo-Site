# Demo Dice: The Presenter-Tron 3000

A steampunk-themed lottery machine for randomly selecting team presentation order during demos.

[Launch Demo Dice](/tools/demo-dice/)

## How to Use

1. **Start**: Click the green lever (or press Enter) to release the balls into the dome
2. **Spin**: Click the red lever (or press Space) to spin and select a ball
3. **Repeat**: Continue spinning until all teams have been assigned a presentation slot
4. **Reset**: Click the green lever again to reset and start over

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| Enter | Start / Reset |
| Space | Spin |
